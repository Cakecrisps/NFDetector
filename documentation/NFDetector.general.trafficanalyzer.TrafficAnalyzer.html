Анализатор сетевого трафика для обнаружения угроз безопасности. Attributes: rules (Dict): Загруженные правила обнаружения угроз из JSON файла. protectips (List[str]): Список защищаемых IP-адресов. stats (Dict): Словарь со статистикой анализа. suspicious_ips (Set[str]): Множество подозрительных IP-адресов. pending_syns (Dict): Словарь для отслеживания полуоткрытых соединений.
