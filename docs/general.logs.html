def is_valid_domain(domain_string: str) -> bool: ¶

Проверяет, соответствует ли строка формату валидного доменного имени. Args: domain_string: Строка для проверки. Returns: True, если строка является валидным доменом, иначе False. Raises: TypeError: если domain_string не является строкой
